# 🚀 DevForge

> A lightweight browser-based code editor inspired by VS Code, built from scratch using HTML, CSS, and JavaScript.

![Version](https://img.shields.io/badge/version-v0.4-blue)
![Status](https://img.shields.io/badge/status-In%20Development-orange)
![License](https://img.shields.io/badge/license-MIT-green)

## 📖 About

DevForge is a personal project built to learn modern web development by creating a real-world application from scratch.

Instead of relying on frameworks or existing editors, the goal is to understand how an IDE works internally by implementing its core features one step at a time.

This project focuses on:

- Clean architecture
- Incremental development
- Real-world software engineering practices
- Learning by building

---

## ✨ Current Features

- 📁 File Explorer
- 📝 Single Editor Architecture
- 🌐 Live HTML/CSS/JavaScript Preview
- 💾 Auto Save

- 📊 Status Bar

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/DevForge.git
```

Open the project:

```bash
cd DevForge
```

Run using VS Code Live Server or any local web server.
