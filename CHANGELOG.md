# DevForge Changelog

## All notable changes to DevForge will be documented in this file.

## v0.5 - Integrated Console

### Added

- Console panel
- Console output capture using `postMessage`
- Support for `console.log()`
- Support for `console.warn()`
- Support for `console.error()`
- Clear Console button
- Auto-scrolling console

### Improved

- Pretty-print objects and arrays
- Multi-line formatting for structured data
- Color-coded log levels

---

## v0.4 - UI Refactor & Status Bar (Current)

### Added

- Status bar
- Dynamic language indicator (HTML/CSS/JavaScript)
- Save status indicator

### Changed

- Refactored from three editors to a single editor
- Introduced `ui` object for cleaner UI management
- Improved project structure
- Cleaner event handling

### Fixed

- HTML not loading after refresh
- File switching bugs
- localStorage loading issues
- Duplicate variable declarations
- JavaScript syntax errors

---

## v0.3 - File Explorer

### Added

- Explorer sidebar
- File switching
- Active file highlighting
- Current file title

### Changed

- Replaced three permanent editors with one dynamic editor

---

## v0.2 - Live Editor

### Added

- HTML editor
- CSS editor
- JavaScript editor
- Live Preview
- Auto Save
- Auto Load using localStorage

---

## v0.1 - Initial IDE

### Added

- Project structure
- Header
- Explorer placeholder
- Editor layout
- Preview panel
- Dark theme
- Responsive grid layout
